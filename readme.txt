Гостевая книга
    --Вывод прошедших модерацию сообщений через пагинацию (отошел от ТЗ и выставил 3 вместо 25Ю очень неохота было сочинять сообщения, значение можно изменить
    тут models/ClassesExt.php (str 9) )
    Поле: Имя (обяз)

Войти
    Тут все просто: вошли - вышли

Личный кабинет (обычный пользователь)
    --Доступен только авторизированным пользователям,
    Поле: имейл
    ПОле: дом.страница
    Поле: пароль
    все поля доступны для редактирования

Личный кабинет (администратор)
    --имеет два дополнительные кнопки
    Кнопка:Модерация сообщений
        --Переход к списку существующих сообщений
            Кнопка: Редактировать сообщение
                --Меню редактирования сообщения
                Поле: текст можно редактировать
                Дроп-меню: выбор статуса активности
                Чекбокс: пометить на удаление
                Кнопка: Редактировать, подтвердить изменения
                Прим. Администратор выполняющий редактуру не можем удалить сам себя
    Кнопка:Список пользователей
        --Переход к списку сущетсвующих пользователей
            Кнопка: Редактировать пользователя
                Поле: имейл
                Поле: дом.страница
                Статус: админ/пользователь
                Чекбокс: удаление

Обращаю внимание что пользователь который оставил сообщения не авторизированным так же записывается в базу и может войти со своими именем и пустым паролем,
а позже сменить его в личном кабинете
